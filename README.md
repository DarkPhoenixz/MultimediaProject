# MultimediaProject
Passaggi per inserire il watermark (DFT-Based)
Caricare l'immagine principale e convertirla in scala di grigi.
Caricare il logo (watermark) e ridimensionarlo per adattarlo.
Applicare la Trasformata di Fourier (DFT) all'immagine.
Modificare la fase del DFT per incorporare il watermark.
Applicare la trasformata inversa (IDFT) per ottenere l'immagine con il watermark nascosto.
